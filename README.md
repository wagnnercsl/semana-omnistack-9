# semana-omnistack-9 - AirCnC
Código da aplicação desenvolvida com base no conteúdo da Semana OmniStack 9.0 (AirCnC)

<h2> Projeto </h2>

Desenvolvido durante a semana OmniStack 9 da Rocketseat, o Aircnc é um projeto que visa conectar empresas que querem abrir spots e desenvolvedores que procuram um lugar para trocar ideias com devs, conhecer a empresa e trabalhar lá por um período.
